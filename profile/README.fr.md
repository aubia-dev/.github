Aubia orchestre vos agents IA de codage (Claude Code, Codex CLI, Mistral CLI, Gemini CLI, Ollama) et les fait se relire mutuellement, pour que vous n'ayez plus à le faire seul un dimanche soir.

## Différenciateurs

- **Cross-review automatique** : chaque sortie d'agent passe par un second LLM d'un provider différent. Verdict tranché, justifications, boucle d'itération jusqu'à 3 fois.
- **Mode Opinions vers Plan** : N agents travaillent en parallèle sur la même tâche, un synthétiseur extrait un plan unifié que vous éditez et exécutez.
- **BYOK strict** : vos contrats Anthropic, OpenAI, Mistral, Google, Ollama restent les vôtres. Aubia n'est pas un revendeur de tokens.
- **Privacy by design** : analytics sans cookies, données isolées en UE par défaut, politique de confidentialité minimaliste.
